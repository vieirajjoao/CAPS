# Documentacao do Projeto

Este diretorio centraliza os documentos tecnicos e organizacionais do projeto CAPS.

## Ordem Recomendada de Leitura

1. `modelagem-e-requisitos.md`
2. `caps-flow.md`
3. `checklist-equipe.md`
4. `entrega-consulta.md`
5. `WORKLOG.md`

## Mapa de Documentos

- `modelagem-e-requisitos.md`
  resumo funcional do sistema, entidades e regras de negocio
- `caps-flow.md`
  organizacao de branches, commits, pushes e pull requests
- `checklist-equipe.md`
  responsabilidades e definicao de pronto por integrante
- `entrega-consulta.md`
  detalhamento da entrega da entidade Consulta
- `WORKLOG.md`
  registro cronologico do que foi executado

## Regra de Manutencao

- manter os documentos coerentes com o estado real da branch
- evitar duplicacao desnecessaria
- registrar mudancas relevantes no `WORKLOG.md`
- revisar links e nomes de arquivos antes de commitar
